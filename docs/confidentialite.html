Confidentialité et données personnelles

Ce qui est stocké

Ton compte (email, mot de passe géré par Supabase) et les données que tu saisis toi-même : offres, fiches entreprises, critères de recherche, CV maître, styles de CV, étiquettes. Tout est hébergé chez Supabase (palier gratuit), protégé par des règles de sécurité (Row Level Security) : personne d'autre que toi ne peut lire ou modifier tes données, même les autres utilisateurs de l'application.

Ta clé IA (Gemini / Anthropic)

Si tu connectes ton assistant IA, sa clé API reste uniquement dans ton navigateur (stockage local) — elle n'est jamais envoyée à notre base de données. Les appels à l'IA partent directement de ton navigateur vers Google ou Anthropic, jamais via un serveur à nous.

Lecture de pages web (offres, LinkedIn)

Ajouter une offre par lien ou chercher des offres LinkedIn peut passer par ton fournisseur IA (qui va lire la page) ou par un relais public tiers gratuit (uniquement le texte de la recherche y transite, jamais tes données personnelles). Aucun serveur à nous n'est impliqué dans cette lecture — voir le code source pour le détail exact.

Tes droits

Depuis l'onglet ⚙️ Paramètres, une fois connecté :

  • Exporter toutes tes données en un clic (fichier JSON).
  • Supprimer définitivement ton compte et toutes tes données, à tout moment, sans avoir à nous contacter.

Mode démo

La démo publique (accessible sans compte) affiche des données 100 % factices — aucune entreprise réelle, aucune donnée personnelle. Rien n'y est jamais enregistré.

Contact

Projet étudiant open-source, gratuit, sans publicité ni revente de données. Le code source est public — n'importe qui peut vérifier ce qui est fait de tes données.
